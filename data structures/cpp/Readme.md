or <br>

run using:<br>
./a.out<br>

##Defination
A standard BFS implementation puts each vertex of the graph into one of two categories:<br>

    Visited<br>
    Not Visited<br>

The purpose of the algorithm is to mark each vertex as visited while avoiding cycles.<br>
